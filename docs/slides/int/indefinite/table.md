# 不定积分表

## 背景补充

### 双曲正弦函数

$y = \sinh x = \dfrac{e^x - e^{-x}}{2}$

反函数

$y = \ln(x + \sqrt{x^2 + 1})$

### 双曲余弦函数

$y = \cosh x = \dfrac{e^x + e^{-x}}{2}$

反函数

$y = \ln(x + \sqrt{x^2 - 1})$

### 双曲正切函数

$y = \tanh x = \dfrac{\sinh x}{\cosh x} = \dfrac{e^x - e^{-x}}{e^x + e^{-x}}$

反函数

$y = \dfrac{1}{2}\ln\dfrac{1 + x}{1 - x}$

---
